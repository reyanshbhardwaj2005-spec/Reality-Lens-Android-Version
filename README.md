# RealityLens – Fake Information Detection App

RealityLens is an application that helps users verify whether information shown in screenshots is real or fake. Users can upload a screenshot directly from their mobile device, and the app sends the image to a verification API for analysis.

The verification API used in this project was developed by my college batchmate, while the Android application, API integration, UI implementation, and overall mobile experience were built by me.

The API analyzes the screenshot content and returns a response indicating whether the information is likely genuine or misleading/fake. The application is designed with a clean and user-friendly interface for quick and efficient fact verification.

## Features

* Upload screenshots from gallery
* Send images to verification API
* Detect real vs fake information
* Clean and modern Android UI
* Fast API response handling
* JSON response parsing

## Tech Stack

* Java
* Android Studio
* Retrofit
* REST API
* JSON Parsing
* Material Design Components

## Use Cases

* Verifying social media posts
* Detecting misinformation
* Fact-checking screenshots
* Educational and research purposes
